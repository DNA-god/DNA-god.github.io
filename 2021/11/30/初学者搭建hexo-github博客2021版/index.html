初学者搭建hexo-github博客2021版

#简介

注:这是一篇关于小白如何搭建hexo博客的文章,全文新手向

1.前期准备:

a.Node.js

​ 下载node.js这是搭建博客很关键的一步

下载地址:https://node.js.org/dist/v4.2.3/node-v4.2.3-x64.msi

b.Git

下载Git(这是搭建博客会经常用到的)

下载地址:https://github.com/git-for-windows/git/releases/dowmload/v2.11.0.windows.3/Git-2.11.0.3-64-bit.exe

c.npm

npm不需要单独下载一个软件，它是node.js下的一款安装工具

e.markdown编辑器

这里笔者推荐给大家的是Typora,这是一款比较简约,操作起来比较快捷的markdown编辑器,还有很多其他的markdown编辑器这里就先不做解释

ps:那么如何确定自己已经安装好了上述搭建工具呢?

首先按住windows+r键

然后在弹出来的运行框内输入cmd

进入黑框页面

在黑框页面下分别敲击node -v,npm -v,git –version

在敲击的下面出现了它的版本号，即为安装成功

e.Hexo下载

在上述的同一个黑框页面下敲击npm install hexo-cli -g进行安装，出现版本号即为安装成功

安装成功后同样需要在黑框页面内敲击hexo -v来验证是否真的安装好hexo

f.Notepad++下载

在官网找到后下载即可

2.创建github账号

为什么要创建github账号？为我们搭建博客的网站建立一个仓库

特别强调,github的账号名和邮箱地址非常关键,一定要记录清楚,后续很多地方都会用到

3.具体步骤:

第一大步:

打开任意一个文件夹,在文件夹下方空白部分点击鼠标右键点开Git Bash Here

在打开的黑框页面内敲击ssh,装一个ssh

下一步敲击:ssh-key-gen -t-rsa -C “邮件地址”

这里的邮件地址就是github账号绑定的邮件地址

敲击完成后在本地电脑里找到.ssh下的id_rsa.pub文件

用Notepad++打开

打开后会出现一串很奇怪的代码，复制代码

将它粘贴到你的github账号里SSH keys的key栏里,保存即可

打开最开始的黑框页面,敲击ssh -T git@github.com测试ssh是否绑定成功

第二大步:

新建一个空白文件夹

在空白处敲击鼠标右键点击git bash here

初始化hexo:在黑框页面敲击hexo init(在这一步可能会出现报错情况，但是没关系，多输入几次就可以了)

完成这一步后,你会发现你的空白文件里出现很多个子文件

然后敲击：hexo s(生成你的hexo页面)

你会看到hexo s下面出现https://localhost:4000

这是你的hexo页面,复制粘贴到浏览器即可查看

第三步:欢迎来到最后一步,真正的,你离成功只有一步之遥

在先前生成的文件里找到_config.yml文件

使用Notepad++打开该文件

在底部按照以下格式进行输入

deploy:

type: git

repository: 这里输入你的github地址

branch: main

保存以上操作后退出即可

点击原先生成的文件空白处,敲击:git bash here

在黑框页面内输入:npm install hexo-deploy-git –save部署相应的工具

敲击:hexo g

随后敲击:hexo d(将文件内的内容上传至github仓库上)

至此你就已经搭建好了一个基础的博客
